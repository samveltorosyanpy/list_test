# CLAUDE.md

Guidance for working in this repo. Comments and user-facing strings are in Russian; keep that convention.

## Что это за проект

Telegram-бот «снайпер» для list.am (армянская доска объявлений). Бот периодически парсит страницу категории на list.am, находит **новые** обычные объявления (от людей, не VIP/ТОП) и мгновенно рассылает ссылки подписчикам в Telegram. Параметры слежки (категория, регионы, цена, валюта, состояние, продавец) настраиваются через Telegram WebApp с фильтрами.

Целевая категория по умолчанию — `62` (недвижимость, продажа).

## Структура

Основной рабочий код живёт в `list_test/`:

- `list_test/bot.py` — **главная точка входа**. Бот на aiogram 3 + асинхронный фоновый цикл слежки. Запускать отсюда.
- `list_test/parser.py` — облегчённый парсер одной страницы list.am. Возвращает `dict[int, str]` (id объявления → полный URL). Импортируется в `bot.py` как `from parser import parse_listam`.
- `list_test/sniper_test.py` — автономный CLI-снайпер (без Telegram), печатает новые объявления в консоль. Более старая/богатая версия парсера: вытаскивает цену/описание/локацию и режет HTML по армянскому маркеру `Սովորական հայտարարություններ`. Полезен как референс и для отладки парсинга.
- `index.html` — Telegram WebApp с UI фильтров. По кнопке шлёт `tg.sendData(JSON)` обратно боту.
- `list_test/user_settings.json` — состояние бота: `{ "url", "subscribers": [chat_id, ...] }`.
- `list_test/listam_seen_ids.json` — кэш уже виденных id объявлений (антиспам).
- `list_test/telegram_subscribers.json` — устаревший формат списка подписчиков (`bot.py` его не использует, источник истины — `user_settings.json`).
- `main.py`, `README.md` — заглушки от шаблона PyCharm, не трогать без причины.

## Запуск

```bash
source .venv/bin/activate
cd list_test
python3 bot.py                  # бот + фоновый цикл слежки
python3 parser.py               # изолированный тест парсера (run_test)
python3 sniper_test.py --once   # одна проверка консольного снайпера
```

Окружение: Python 3.14, venv в `.venv/`. Ключевые зависимости: `aiogram` 3.29, `curl_cffi` (обход анти-бот защиты через `impersonate="chrome120"`), `beautifulsoup4`, `requests`.

Токен бота читается из `os.getenv("TELEGRAM_BOT_TOKEN", <fallback>)`. **В `bot.py` сейчас захардкожен реальный токен как fallback — при коммитах его нельзя светить, лучше вынести в переменную окружения.**

## Как работает поток данных

1. Пользователь жмёт `/start` → его `chat_id` добавляется в `subscribers`.
2. Пользователь настраивает фильтры в WebApp (`index.html`) → бот получает JSON в хендлере `F.web_app_data`.
3. `build_listam_url(filters)` собирает URL list.am из полей фильтра (см. контракт ниже) и сохраняет в `user_settings.json`.
4. `background_sniper_loop()` каждые `POLL_SECONDS` (180 с) парсит текущий URL через `parse_listam`, который крутится в executor-потоке, чтобы не блокировать event loop.
5. Новые id (которых нет в `seen`) рассылаются подписчикам, затем кэш `seen` обновляется. На самом первом запуске (`is_first_run`) рассылка подавляется, чтобы не спамить старой выдачей.

### Контракт WebApp → bot (`build_listam_url`)

JSON-поля из `index.html`: `category_id`, `regions_ids` (список → параметр `n`), `currency_id` (`crc`), `min_price` (`price1`), `max_price` (`price2`), `cnd_id` (`cnd`), `user_id` (`user`). Плюс `*_text` / `regions_names` — только для красивого отчёта пользователю, в URL не идут. При изменении ключей правь обе стороны: `index.html` и `build_listam_url`.

## Особенности парсинга (важно)

- list.am отдаёт несколько блоков `div.gl`: VIP/ТОП идут первыми, обычные объявления — **последними**. `parse_listam` берёт `gl_containers[-1]` и тянет ссылки `^/item/\d+`. Если вёрстка сайта изменится, ломается именно здесь.
- `sniper_test.py` использует другой подход — режет HTML по армянскому маркеру `Սովորական հայտարարություններ`. Это запасная стратегия, если подход «последний `.gl`» перестанет работать.
- В `parser.py` есть отладочный `print(soup)` (строка ~39), который вываливает весь HTML в stdout. Под прод его надо убрать.
- Пустой результат `{}` обычно значит: list.am заблокировал IP, сменилась вёрстка, либо на странице нет обычных объявлений.
